AI Chatbot using RAG and Ollama – Developed an AI-powered chatbot using LangChain, Ollama, and Streamlit to provide context-aware responses from PDF documents. Implemented Retrieval-Augmented Generation (RAG) with FAISS vector indexing and Sentence Transformers embeddings for efficient semantic search. Integrated PyPDF for document processing and Hugging Face embedding models to improve retrieval accuracy and response quality.
